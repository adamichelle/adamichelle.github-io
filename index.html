About Me

I'm a front end developer based in Lagos, Nigeria.
I started web application development in 2016 during my internship at a software development firm. At the time, PHP and MySQl were my tools of development. More recently, I discovered my love for JavaScript as I took and finished courses in Udacity and CodeCademy as a way to improve myself and learn more skills.

My HTML and CSS skills are very good and I'm conversant with the use of JavaScript and JQuery. I am also very familiar with web responsiveness tools such as Bootstrap and can implement responsiveness in sites from scratch if needed.
I am currently working on learning a number of front end frameworks to improve my front end JavaScript skills.

I am passionate about building web apps to provide solutions to business and people who need them. You can check out some of my projects below.
