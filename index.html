About Me


I'm a front end developer based in Lagos, Nigeria.
I started web application development in 2016 during my internship at a software development firm. At the time, PHP and MySQl were my tools of development. My HTML and CSS skills are very good and I'm conversant with the use of JavaScript and JQuery. I am also very familiar with web responsiveness tools such as Bootstrap and can implement responsiveness in sites from scratch if needed.
